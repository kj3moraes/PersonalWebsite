001 : verCE

JIT Compilers

Just-In-Time compilers are compilers that will compile the source code at run-time. It is a hybrid between AOT (Ahead-Of-Time) compilers and Interpreters. JIT compilers fix the cons of both static compilers and interpreters and offer enormous avenues for development. Since AOT compilers generate object files that are then linked to the executable, they are not very flexible. You cannot write a whole program in one stroke with interpreters. JIT compilers solve both these issues. I strongly believe that JIT compilers are the way forward for modern programming languages.

verCE and its Features

verCE is my custom cross-platform JIT compiler. I designed it using the Kaleidoscope tutorial on the LLVM website but I integrated several features and development strategies that I learnt from CS241 at the University of Waterloo. This project was a great learning experience for me and I am very proud of it. The programming language offers the following features :

  1. `extern` keyword
  2. if/for control statements
  3. double variables
  4. global statements
  5. comments

The compiler was developed with the following design properties

  1. Visitor Design Pattern - for code generation, to decrease coupling between code generation and abstract syntax tree
  2. Smart Pointers - eliminate the need for explicit memory management
