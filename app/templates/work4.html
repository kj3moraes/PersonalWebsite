004 : Battleship Game Engine

This project is a text-based Battleship game engine that I built in Java. It supports 3 different engines that are in increasing level of difficulty.

  • Easy Engine : This engine will play randomly and hit any targes without registering the hits it makes. It is a rather naive gameplay that will take about 80 moves for a human to complete.
  • Smart Engine : This engine employs a Hunt and Target algorithm to play just like a human would. The Hunt-and-Target mechanism registers the hits and like a human, it will start to sequentially target the remaining squares. An additional improvement to build in a heuristic for targeting most used squares.

I developed this project to experiment more with Java and Object Oriented Programming concepts. Battleship is also a game that I have enjoyed playing in the past and I wanted to build a game engine that I could play with if I didn't have any other. You can discover the project here Github
